
# 🍽️ Restaurant App

A modern Full Stack Restaurant Management Web Application that allows users to browse food items, place orders, and manage restaurant operations efficiently.

---

## 🚀 Features

- User Registration & Login
- Browse Food Menu
- Search Food Items
- Food Categories
- Add to Cart
- Place Orders
- Order History
- Responsive User Interface
- Admin Dashboard
- Menu Management
- Order Management

---

## 🛠️ Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript
- React.js

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Version Control
- Git
- GitHub

## ⚙️ Installation

### Clone the repository

```bash
git clone git@github.com:joshi-0706/restaurant-app.git
```

### Navigate to project

```bash
cd restaurant-app
```

### Install dependencies

```bash
npm install
```

### Start the application

```bash
npm start
```
