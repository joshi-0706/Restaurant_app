## ⚙️ Installation

### Clone the repository

```bash
git clone git@github.com:joshi-0706/restaurant-app.git
```

### Navigate to project

```bash
cd restaurant-app
```

### Install dependencies

```bash
npm install
```

### Start the application

```bash
npm start
```
