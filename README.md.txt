# 🍽️ Restaurant App

A Full Stack Restaurant Management Web Application built using the MERN Stack.

## 📌 Features

- User Authentication
- Browse Restaurant Menu
- Search Food Items
- Add to Cart
- Place Orders
- Order History
- Admin Dashboard
- Menu Management
- Responsive Design

## 🛠️ Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- Git & GitHub
